## 🚀 Getting started

For development, **Bun is required**.

1. 🧑‍🤝‍🧑 Clone the repo
```bash
gh repo clone
Scolup/Linkdirecte
```
2. ⬇️ Install deps
```bash
bun install
```
3. ⛏️ Start working!
